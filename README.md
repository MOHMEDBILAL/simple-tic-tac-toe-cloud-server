## Overview

This is a simple Tic Tac Toe web application hosted on InfinityFree using Infrastructure as a Service (IaaS) principles. It demonstrates basic web hosting, HTML/CSS/JS implementation, and functionality testing.

## Setup Steps (for replication)

1. **Create InfinityFree Account**

   * Go to [https://infinityfree.net](https://infinityfree.net)
   * Sign up with an email address.
2. **Create a Hosting Account**

   * Click **Create Account**
   * Choose a free subdomain (e.g., simpletictactoe.infinityfreeapp.com)
   * Wait for account activation.
3. **Open File Manager**

   * Navigate to `/htdocs` folder.
   * Delete any default `index.html` file if present.
4. **Upload Website**

   * Upload your `index.html` file into `/htdocs`.
5. **Test Website**

   * Visit your domain: [http://yourdomain.infinityfreeapp.com](http://yourdomain.infinityfreeapp.com)
   * Check that the Tic Tac Toe game works correctly.

## Game Instructions

1. Click on any empty cell to place your X or O.
2. Players alternate turns automatically.
3. The game announces the winner or a draw.
4. Click **Reset Game** to start a new game.

## Script Explanation

* The JavaScript logic is included inside `index.html`:

  * Tracks player moves in an array.
  * Checks all win conditions after each move.
  * Updates the on-screen status text dynamically.
  * Handles game reset functionality.
